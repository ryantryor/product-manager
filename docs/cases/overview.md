## 如何贡献案例

欢迎提交 PR 分享你的 AI 产品设计案例。每个案例建议包含：

1. **产品背景**：产品是什么，解决什么问题
2. **设计挑战**：面临的核心设计问题
3. **解决方案**：具体的设计方案和思路
4. **效果验证**：上线后的数据表现
5. **经验总结**：可复用的经验和教训
