# AI 产品需求分析

## AI 需求评估框架

不是所有需求都适合用 AI 来解决。使用以下框架来评估：

### 适合 AI 的场景

| 特征 | 说明 |
|------|------|
| 模式识别 | 从大量数据中发现规律 |
| 自然语言处理 | 理解、生成、翻译文本 |
| 个性化推荐 | 根据用户偏好定制内容 |
| 重复性判断 | 需要大量人工审核的任务 |
| 创意辅助 | 头脑风暴、内容生成 |

### 不适合 AI 的场景

| 特征 | 说明 |
|------|------|
| 需要 100% 准确 | 如金融交易执行 |
| 规则明确 | 简单 if-else 就能解决 |
| 数据不足 | 没有足够的训练/参考数据 |
| 强监管 | 法律法规要求可追溯的确定性决策 |
