# 产品经理必读书目

> 从入门到精通，覆盖产品思维、用户研究、数据驱动、商业战略和 AI 产品的完整书单。

## 产品思维与方法论

### 入门必读

| 书名 | 作者 | 推荐理由 |
|------|------|---------|
| 《启示录：打造用户喜爱的产品》 | Marty Cagan | **PM 圣经**，讲清楚产品经理到底做什么 |
| 《人人都是产品经理》 | 苏杰 | 中文 PM 入门经典，接地气的实战指南 |
| 《产品方法论》 | 俞军 | 俞军（百度前产品 VP）的产品哲学，效用模型深刻 |
| 《用户体验要素》 | Jesse James Garrett | 5 层模型（战略→范围→结构→框架→表现），PM 必知框架 |
| 《Don't Make Me Think》 | Steve Krug | 可用性设计入门，200 页改变你对产品的看法 |

### 进阶提升

| 书名 | 作者 | 推荐理由 |
|------|------|---------|
| 《INSPIRED：产品管理如何打造用户喜爱的产品》 | Marty Cagan | 启示录升级版，强调产品发现和验证 |
| 《Shape Up》 | Basecamp | 挑战 Scrum 的产品开发方法论，6 周周期，免费在线阅读 |
| 《持续发现习惯》 | Teresa Torres | 机会解决方案树（OST），持续做产品发现 |
| 《产品心经》 | 闫荣 | 腾讯系产品方法论，B端C端都有覆盖 |
| 《幕后产品》 | 王诗沐 | 网易云音乐前产品负责人，真实产品决策案例 |

## 商业与战略

| 书名 | 作者 | 推荐理由 |
|------|------|---------|
| 《创新者的窘境》 | Clayton Christensen | **颠覆式创新**理论，理解技术变革如何改变市场 |
| 《跨越鸿沟》 | Geoffrey Moore | 技术产品从早期采用者到主流市场的跨越策略 |
| 《竞争战略》 | Michael Porter | 波特五力模型，理解竞争格局 |

| 《定位》 | Al Ries & Jack Trout | 营销和品牌定位经典，PM 理解市场的基础 |
| 《从零到一》 | Peter Thiel | PayPal 创始人讲创新和垄断，思维高度拉满 |
| 《商业模式新生代》 | Alexander Osterwalder | 商业模式画布（BMC），一页纸看清商业本质 |
| 《平台革命》 | Parker, Van Alstyne | 平台型产品的网络效应和竞争策略 |

## 推荐阅读路径

### 🟢 入门（0-1 年 PM）

```
启示录 → 用户体验要素 → Don't Make Me Think → 精益创业 → 人人都是产品经理
```

### 🔵 进阶（1-3 年 PM）

```
INSPIRED → 精益数据分析 → 增长黑客 → 创新者的窘境 → 金字塔原理
```

### 🟣 高阶（3-5 年 PM）

```
产品方法论 → 跨越鸿沟 → 平台革命 → 竞争战略 → 赋能
```

### 🔴 AI 产品经理专项

```
这就是ChatGPT → AI产品经理 → Designing ML Systems → Building LLM Apps → 算法霸权
```

---

::: tip 阅读建议
- **每月至少读 1 本**，坚持一年就能建立完整知识体系
- **做笔记和实践**，光读不练等于没读
- **优先读英文原版**，翻译有时会丢失精髓
- **结合工作场景**，带着问题读书效率最高
:::
