# AI 工具链概览

作为 AI Native 产品经理，你需要掌握一套完整的 AI 工具链来提升工作效率和产品认知。

## 工具矩阵

### 基础 AI 能力

| 工具 | 用途 | 推荐场景 |
|------|------|----------|
| Claude | 深度分析、长文写作、代码理解 | 需求分析、PRD 撰写、技术方案评审 |
| ChatGPT | 通用对话、头脑风暴 | 创意发散、快速问答 |
| Gemini | 多模态理解 | 图片分析、视频理解 |

### 产品设计

| 工具 | 用途 |
|------|------|
| v0 / Bolt | AI 生成 UI 原型 |
| Figma + AI 插件 | 设计稿生成与优化 |
| Cursor / Claude Code | AI 辅助产品原型开发 |

### 效率工具

| 工具 | 用途 |
|------|------|
| Notion AI | 文档协作与知识管理 |
| AI 搜索引擎 | 竞品调研与信息检索 |
| AI 会议助手 | 会议纪要与待办提取 |
