## 这是什么

一个系统化的 **AI Native 产品经理知识库**，覆盖从技术基础到产品战略的完整知识体系。无论你是刚转型 AI 方向的产品经理，还是希望系统提升 AI 产品能力的从业者，都能在这里找到实用的框架、方法论和实战经验。

> AI Native 不是「产品 + AI」，而是「AI × 产品」——用 AI 思维重新定义产品设计。

## 核心亮点

| 特色 | 说明 |
|------|------|
| **体系完整** | 从技术基础 → 产品设计 → 方法论 → 战略 → 伦理，覆盖 AI PM 全链路 |
| **产品经理视角** | 技术内容聚焦"对产品的影响"，不是教你写代码 |
| **实战导向** | 包含成本估算、选型对比、面试攻略等实操内容 |
| **持续更新** | 跟踪 AI 领域最新发展，持续补充新内容 |

## 快速开始

```bash
# 克隆仓库
git clone https://github.com/ryantryor/product-manager.git
cd product-manager

# 安装依赖
npm install

# 本地启动
npm run docs:dev
```

浏览器打开 `http://localhost:5173/product-manager/` 即可阅读。

## 推荐学习路径

```
Level 1: AI 认知入门（1-2 周）
  → 入门指南 → AI 技术基础概览 → LLM 深度理解

Level 2: 工具实践（2-4 周）
  → AI 工具链 → Prompt Engineering → AI 设计工具

Level 3: 方法论建立（4-8 周）
  → 产品方法论 → AI 产品设计 → 产品战略

Level 4: 实战进阶（持续）
  → 实战案例 → 四层架构 → 模型选型 → 面试攻略
```
