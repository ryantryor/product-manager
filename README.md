## 这是什么

一个关于**如何成为 AI Native 产品经理**的知识库，系统梳理 AI 时代产品经理需要掌握的认知、工具、方法论和实战经验。

> AI Native 不是「产品 + AI」，而是「AI × 产品」——用 AI 思维重新定义产品设计。

## 内容结构

```
📖 入门指南          什么是 AI Native PM、与传统 PM 的区别、学习路径
🛠️ AI 工具链        LLM 基础认知、Prompt Engineering、AI 设计工具
📐 产品方法论        AI 需求分析、功能设计模式、数据驱动决策
🚀 实战案例          真实 AI 产品设计案例拆解与经验总结
```

## 快速开始

```bash
# 克隆仓库
git clone https://github.com/ryantryor/product-manager.git
cd product-manager

# 安装依赖
npm install

# 本地启动
npm run docs:dev
```

浏览器打开 `http://localhost:5173/product-manager/` 即可阅读。
